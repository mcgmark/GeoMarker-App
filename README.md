    <h2>What?</h2>
    <p>GeoMarker is an app created to allow you to post and view a location of something on a map. 
    </p>

    <h3>What should I mark?</h3>
    <ul>
        <li>Location of a hidden item.</li>
        <li>A location special to you.</li>
        <li>Where you lost or found something.</li>
        <li>Roadside attraction you found on your last roadtrip.</li>
        <li>Post your weekend garage sale.</li>
        <li>Keep track of bigfoot sightings.</li>
        <li>Anything you want, but family friendly!.</li>
    </ul>

    <h2>Why?</h2>
        <p>
            This app is the semester assignment for Georgian College Comp2084 Server-Side Scripting class. This app is a demonstration of the knowledge and skills 
            obtained through the semester.
         </p>

    <h2>What else?</h2>
        <p>
           This was built on the ASP.Net framework and implements user input, user authentication, MS SQL database CRUD, and Map API. This project also implements Javascript and Bootstrap.
        </p>
